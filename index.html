Qu'est ce que le développement web

Le développement Web est l'ensemble des activités liées à la création de sites web et applications web, de la maquette du site jusqu'à la mise en ligne, en utilisant des langages de programmation, des technologies et des outils appropriés pour donner vie à des expériences interactives sur Internet.

Le HTML, la structure de tout projet

Le HTML est la base de tout projet, c'est un langage de balisage utilisé pour décrire la structure et le contenu d'une page Web. Il fournit une base solide sur laquelle le développeur peut se concentrer afin de créer son site web en utilisant un ensemble de balises prédéfinies.

  • <!DOCTYPE html>, la tout première balise qu'on voit dans un dossier, elle signifie que nous travaillons bien sur un fichier HTML.
  • <html> : Elle définit l'élément racine du document HTML. Tout le contenu de la page Web se trouve à l'intérieur de cette balise.
  • <head> : Elle contient les informations non visibles de la page web, telles que le titre de la page ou les liens vers les fichiers externes.
  • <body> : Elle englobe tous le contennu visible de la page web : textes, images, les liens et autres.
  • <link> : Elle contient le liens vers les fichiers externes.

Architecture, les balises html

  • La balise <header>, qui signifie "entête", est une balise dans laquelle nous allons mettre tout ce qui se tiendra en haut de la page, comme la barre de navigation par exemple.
    La balise <footer> elle, représente le pied de page, tout ce qui sera tout en bas de la page, après le contenu principal.
  • La balise <section>;, celle qu'on utilisera pour englober de grosses parties de la page, elle est souvent utilisée comme conteneur, un peu comme une boîte.
    La balise <div>;, elle a la même utilité que la balise <section>, cependant elle sert surtout a englober les sous-parties, par exemple dans une <section>;, si nous en avons besoin, nous y mettrons une div pour mieux structurer la page.
  • La balise <p>, très simple, c'est une balise dans laquelle nous y mettrons du texte, "p" comme "paragraphe".
    La balise <span> elle, est une balise servant à cibler une partie de texte spécifique qu'on voudrait changer (de couleur, ou de taille par exemple)
  • Puis nous avons enfin la balise <ul>, suivi de la balise <li>, les listes, qui servent à ordonner certains de nos textes, nous pouvons les changer sous format de liste afin de mieux s'y retrouver lors de la lecture du site.

Donner du style à son site facilement avec CSS

  • Tout d'abord, les bases, savoir lier son fichier CSS à son fichier HTML. Grâce à la balise <link>, nous pouvons lier notre fichier CSS à notre fichier HTML afin de pouvoir modifier son style.
  • En CSS, c'est très simple, il nous suffit simplement de cibler une balise HTML, <p> par exemple, puis lui donner des propriétés afin de changer sa couleur, sa taille ect...
  • Passons aux propriétés ainsi qu'aux valeurs.
    La propriété en CSS c'est simplement la chose que nous voulons modifier sur notre élément, la couleur par exemple (color en CSS), ou même la taille (font-size) et plus encore
    Et la valeur CSS c'est la façon dont nous voulons modifier cette propriété, pour la couleur par exemple nous mettrons le rouge ( color : red; )
